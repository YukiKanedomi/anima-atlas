# 重い点はどこにある？——位相の話

ここまでは「どれだけ揺れるか（大きさ）」を見てきました。でも回転機械では、もう一つ大事な情報があります。**揺れがどちらを向いているか**——専門的には「位相」です。

円板には重心のずれ（偏心）があり、そこが一番重い点（**重い点**）です。ゆっくり回しているときは、ロータは重い点が外を向いた向きにふくらみます。ところが回転を上げて危険速度を超えると、**揺れの向きが重い点と正反対（180°）にひっくり返る**のです。

この向きのズレ（位相 $\varphi$）も、回転数の比 $r = \Omega/\omega_n$ で決まります。

$$\varphi = \operatorname{atan2}\!\left(2\zeta r,\; 1 - r^2\right)$$

下のグラフは、$r$ に対して位相 $\varphi$ がどう変わるかを示します。スライダーで減衰を変えながら、危険速度（$r = 1$）でちょうど **90°** を通って、最後は **180°** に近づく様子を見てください。

この「180°の反転」は、前の節で出てきた**自動調心**（高速で揺れが落ち着く）の正体でもあります。揺れが重い点の反対側に来ることで、結果としてロータは自分の重心を回転中心に近づけていくのです。下のアニメで、実際に見てみましょう。

実務では、この位相はとても役に立ちます。機械の振動を測ったとき、位相が 90° 付近かどうかで「いま危険速度の近くにいるか」を判断でき、バランス取り（おもりをどこに足すか）の手がかりにもなります。

**模式・出典について**：位相のグラフは減衰のある1自由度系の定常応答の位相 $\varphi = \operatorname{atan2}(2\zeta r,\,1-r^2)$ をそのまま描いたものです。振れ回りアニメは、同じ $R/e$ と $\varphi$ の関係を上から見た**模式**で、偏心 $e$ は見やすく誇張しています（物理関係は式どおり）。
