# 前向き？後ろ向き？——振れ回りの向き

ここまで「揺れの大きさ」と「位相」を見てきました。次は、振れ回りそのものの**形と向き**に注目します。

振れ回りには**向き**があります。軸の中心が、回転と**同じ向き**に回るのが **前向き振れ回り（forward whirl）**、**逆向き**に回るのが **後ろ向き振れ回り（backward whirl）** です。前向きはよくある普通の状態ですが、後ろ向きは支持の異方性やジャイロ効果で現れ、見落とすと危険なこともあります。

便利なことに、どんな平面の振れ回りも、**前向きの円**と**後ろ向きの円**の足し合わせで表せます。前向き成分を $F$、後ろ向き成分を $B$ とすると：

$$x(t) = (F+B)\cos\theta,\qquad y(t) = (F-B)\sin\theta$$

この2つの比で、軌道の形と向きが決まります。下のスライダーで $F$ と $B$ を動かして確かめてください。

- $B = 0$（前向きだけ）→ **円**（前向き）
- $F,\ B$ どちらもある → **楕円**（支持が硬軟あるとこうなる）
- $F = B$ → **直線**（往復運動）
- $F < B$ → 楕円のまま**向きが後ろ向き**に反転

支持（軸受や台）の硬さが縦と横で違うと、ロータは円ではなく**楕円**を描きます。これは「軸受の異方性」が軌道に出ているサインで、現場で軌道（オービット）を見れば支持の素性が読み取れます——次の節では、その軌道をどうやって測るかを見ていきます。

**模式・出典について**：上の図は平面振れ回りの標準的な前向き／後ろ向き分解（$z(t)=F e^{i\omega t}+B e^{-i\omega t}$）をそのまま動かした**模式**です。寸法は見やすさのためのスケールで、形と向きの関係は式どおりです。
