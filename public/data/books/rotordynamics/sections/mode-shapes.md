# 軸はいくつもの形で揺れる——曲げモード形状

ここまで、ロータをだいたい「1つのばねの上の円板」として扱ってきました。危険速度も1つ。でも本物の軸は**連続体**——細長い1本の棒です。棒は、1つの形だけでなく、**いくつもの違う形**で揺れることができます。その一つひとつが**モード**で、それぞれに固有の危険速度があります。

下で**モード**を 1次・2次・3次と切り替えてみてください。軸のしなり方が変わります。

:::mode-shape
{
  "title": "モードを切り替える：高いモードほど節が増える",
  "maxMode": 3,
  "boundaryDefault": "pinned",
  "caption": "赤い点が「節」＝揺れても動かない点。n次モードは節を (n−1) 個持ちます。単純支持では危険速度の比は 1 : 4 : 9。"
}
:::

ポイントは2つです。

- **モードが上がるほど「節」が増える**。節（ふし）は、揺れているのに**動かない点**。1次は節ゼロ（真ん中が一番大きく振れる）、2次は中央に1つ、3次は2つ。バイオリンの弦と同じ仕組みです。
- **危険速度はモードごとに違う**。単純支持の軸では、固有振動数は $n^2$ で上がっていきます（**1 : 4 : 9**）。だから1次の危険速度を無事に通過しても、もっと上の回転数で**2次・3次の危険速度**が待っています。実機で「何次まで考えるか」が設計課題になるのはこのためです。

## 支持が変わると、形も危険速度も変わる

モード形状は**端の支え方（境界条件）**で決まります。`支持` を **自由-自由** に切り替えてみてください。

軸受が硬ければ軸の端は「単純支持」（ピン）に近く、モードは $\sin$ の形。一方、**やわらかい軸受の上に乗ったロータ**は、軸そのものは両端**自由**に近くなります。すると——

- 1次モードでも軸の内側に**2つの節**ができる（多くの場合、ちょうど軸受の近く）。
- 危険速度の比も $1 : 4 : 9$ ではなく **$1 : 2.76 : 5.40$** に変わります。

同じ軸でも、**支えが硬いか柔らかいかで揺れの形が別物になる**。これがモード解析の出発点であり、軸受の設計がロータ全体の挙動を左右する理由です。

:::deep なぜ「節」と「振動数の比」が決まるのか
細い棒の曲げ振動は、はり（Euler–Bernoulli）の式で記述され、モード形状 $\psi(\xi)$（$\xi=x/L$）は境界条件で決まります。

- **単純支持**：$\psi_n(\xi)=\sin(n\pi\xi)$。$\beta L = n\pi$。固有振動数は $\omega_n \propto (\beta L)^2 = (n\pi)^2$ なので比は $n^2 = 1:4:9$。
- **自由-自由**：$\cosh\beta L\cdot\cos\beta L = 1$ を満たす $\beta L \approx 4.730,\ 7.853,\ 10.996$。形は
  $$\psi_n(\xi)=\cosh\beta L\xi+\cos\beta L\xi-\sigma_n\,(\sinh\beta L\xi+\sin\beta L\xi)$$
  振動数比は $(\beta L_n/\beta L_1)^2 \approx 1:2.76:5.40$。

要するに「**振動数 ∝ (波数)²**」。波が細かいほど（高次ほど）急激に振動数＝危険速度が上がります。節の位置と数も、この $\psi_n$ のゼロ点として決まります。
:::

モード形状が読めると、「この危険速度では軸のどこが大きく振れるか」「どこにセンサや釣り合いおもりを置くと効くか」が見えてきます。次章以降のバランシングや軸受設計は、この**モードの地図**の上で行われます。

**模式・出典について**：図は Euler–Bernoulli はりの標準的なモード形状をそのまま描いた**模式**です。単純支持は $\sin$、自由-自由は上式（$\beta L$ は丸め値）。節の位置・数、振動数比はすべて式どおりで、振幅は見やすさのための表示スケールです。
