# 規格というものさし——ISOとAPIの地図

ここまで、振動を**測り**（波形・スペクトル）、**見渡し**（ウォーターフォール）てきました。でも現場には、もう一つ欠かせない問いがあります——**「で、これは合格なの？」**。

「揺れが大きい／小さい」は主観です。新品なのか、もう寿命なのか。発注した機械が契約どおりの品質なのか。こうした判断を、誰が測っても同じ結論になるよう取り決めたのが**規格**——いわば、世界共通の**ものさし**です。

回転機械の規格は、ロータの**一生のどの局面か**で使い分けます。下の地図で、局面を切り替えてみてください。

## 大きく2つの問い

規格はたくさんありますが、ロータダイナミクスでまず押さえるべきは**2つの問い**です。

- **作る前に：どれだけ釣り合わせる？** → **ISO 21940-11**。残留不釣り合いの許容量を、**品質等級 G** と回転数から決めます。「G2.5」「G6.3」といった等級は、許容できる偏心の大きさ（×回転数）を表します。
- **動かしてから：この揺れは大丈夫？** → **ISO 20816**。測った振動の実効値が、**ゾーン A〜D** のどこに入るかで評価します。

そしてこの土台の上に、**機種ごとの約束**（API 610/617/612 など）が乗ります。

## ゾーン A〜D という考え方

ISO 20816 の評価は、振動の実効値を4つの帯で区切ります。考え方はとてもシンプルです。

- **ゾーン A**：新品・出荷時に期待される、低い振動。
- **ゾーン B**：長期運転して問題ない範囲。
- **ゾーン C**：長期連続運転には不適。短期なら許容、早めに対策を。
- **ゾーン D**：損傷を起こしうる危険な水準。

境界の具体的な数値（mm/s など）は、機械の種類・大きさ・支持の硬さによって変わります。**その「ものさしの目盛り」を実際に動かして読むゲージは、次の専用ブロックで**扱います。

## 品質等級 G という考え方

ISO 21940-11 の釣り合い品質等級は、**許容残留偏心 × 角速度 = 一定（＝G）**という関係でできています。

$$e_{per} \cdot \Omega \approx G \times 1000 \quad (\text{単位：} \mathrm{mm/s})$$

つまり同じ等級 G でも、**回転数が高い機械ほど、許される偏心は小さく**なります。高速機ほど厳しく釣り合わせなければならない、という現場の感覚が、この一本の式に表れています。等級の選び方と許容量の計算は、**第7章バランシングの続き**で具体的に扱います。

:::deep 番号の歴史——なぜ似た規格がいくつもあるのか
規格は改訂で番号が変わったり、統合されたりします。混乱しやすいので、対応を押さえておきましょう。

- **振動評価**：かつては「回転軸の振動」を **ISO 7919**、「非回転部（軸受台）の振動」を **ISO 10816** と、別々に定めていました。2016年からは両者を統合した **ISO 20816** に再編されています。古い図面や論文で 7919／10816 を見たら、いまの 20816 に対応すると考えてください。
- **釣り合い**：剛性ロータの釣り合い品質等級は、長く **ISO 1940-1** として知られていました。現在は **ISO 21940-11** に置き換わっています（中身の考え方＝等級 G は引き継がれています）。
- **ISO と API の関係**：ISO は国際的な共通土台、API（米国石油協会）は石油・ガス機械向けの、より具体的で厳しい調達仕様。実務では「ISO を土台に、API や各社仕様で上乗せ」という重ね方をします。
:::

**出典・注意について**：本節は規格の**全体像をつかむための概観**です。ゾーンの境界値や等級の許容量といった具体的な数値、適用範囲（機械の種類・パート番号）は規格ごと・改訂ごとに定められており、有料で頒布されています。実際の判定・設計には、必ず**各規格の最新版と該当パート**を参照してください。ここでは番号と役割の地図を示すにとどめ、数値の読み取りは続く専用ブロック（評価ゲージ・等級セレクタ）に譲ります。
