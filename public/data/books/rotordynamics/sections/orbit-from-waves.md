# 軌道は2つの波からできている

現場でロータの振動を測るときは、**変位センサ**を軸の近くに置いて「ある方向の動き」を時間で記録します。出てくるのが、いかにも振動らしい**波形**です。

センサを2つ——**横方向**と**縦方向**——に置けば、横の波 $x(t)$ と縦の波 $y(t)$ の2本が得られます。そしてこの2本を、横軸 $x$・縦軸 $y$ に取り直して重ねて描くと……前の節で見た**軌道**になります。

つまり、**軌道とは「2つの時間波形を組み合わせたもの」**。下で前向き成分 $F$・後ろ向き成分 $B$ を動かすと、上の2つの波形と下の軌道が**同時に**変わります。波と軌道の対応を、目で確かめてください。

動かすと、こんな対応が見えてきます。

- $B = 0$：横と縦が**同じ大きさで90°ずれた波** → 軌道は**円**
- $F = B$：縦の波 $y(t)$ が**消えて**横の波だけ → 軌道は**直線**（往復）
- その間：縦と横で大きさが違う → 軌道は**楕円**

「波形」と「軌道」は同じ運動の2つの見方にすぎない——この行き来ができると、現場の波形からロータの動きが読めるようになります。

**模式・出典について**：波形は $x(t)=(F+B)\cos\theta,\ y(t)=(F-B)\sin\theta$ をそのまま描いた**模式**で、振幅は見やすさのためのスケールです。波と軌道の対応関係は式どおりです。
