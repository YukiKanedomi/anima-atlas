# 回転座標系で見る——なぜ回る軸は自重で疲労するのか

ここまでは、地面に立った私たち（**静止座標系**）から振れ回りを眺めてきました。でも視点を変えて、**軸といっしょに回る座標系**から同じ現象を見ると、まったく違う顔が見えてきます。そしてその違いが、回転機械の設計でいちばん大事な問いの一つ——**どの応力が疲労を生むか**——に直結します。

回る軸の表面に、小さなひずみゲージを貼ったと想像してください。ゲージは軸といっしょに角速度 $\Omega$ で回っています。ゲージにとっての「世界」は、回転座標系で見た世界です。ここで2つの状況を比べます。

## 状況1：不釣り合いによる同期振れ回り

不釣り合いの振れ回りは、回転と**同じ速さ・同じ向き**（同期）で回ります。たわみのベクトルは、いつも重い点と同じ側を向いたまま $\Omega$ で回り続ける。つまり**軸といっしょに回るゲージから見ると、たわみは止まって見えます**。曲がってはいるけれど、動かない。

ゲージが感じる曲げ応力は**一定（直流）**。向きも大きさも変わらないので、**振動する応力にはなりません**。だから「1×の同期振れ回りそのもの」は、軸の材料を疲労させにくいのです。これは直感に反します——あんなに回っているのに。

この円に合わせて自分も $\Omega$ で回ると、点はピタリと止まります。動かない＝応力が変化しない、ということ。

## 状況2：一定の横向き荷重（自重・片寄り）

今度は、**静止座標系で見て一定**の横向きの力——たとえば横置きの軸にかかる**重力**を考えます。地面から見れば、力はいつも真下を向いていて動きません。

ところが、回るゲージから見ると話は逆転します。**静止系で止まっている下向きの力は、回転系では $-\Omega$ で逆回りして見える**のです。ゲージは1回転するあいだに、引っ張り → 圧縮 → 引っ張り…と、**1回転あたり1往復の完全反転応力**を受けます。これが回転数 $\Omega$（＝1×）の交番応力、いわゆる**回転曲げ**です。

**静止系で一定の荷重 → 材料は 1× の完全反転応力（疲労の元）。**

横置きの軸が自重で疲れていくのは、荷重が変動するからではありません。**荷重は止まっていて、材料のほうが応力の場の中を回り抜けていく**から。視点を回転系に移して初めて、その交番が見えます。

:::deep 座標変換でひとことで言うと
静止系のたわみを複素数 $z = x + iy$ で表し、回転系のたわみを $\zeta$ とすると、両者は角度 $\Omega t$ だけ回した関係です。

$$\zeta = z\, e^{-i\Omega t}$$

- **同期振れ回り**：$z = R\,e^{i\Omega t}$ を入れると $\zeta = R$（定数）。回転系で**止まる**。
- **静止系で一定の荷重によるたわみ**：$z = \delta$（定数）を入れると $\zeta = \delta\,e^{-i\Omega t}$。回転系で**$-\Omega$ で回る**＝材料は 1× の交番応力。

同じ $\Omega$ でも、片方は直流・片方は交番。**しんどさは、どの座標から見るかで決まる**わけです。
:::

## だから何が嬉しいか

この視点の切り替えは、現場の謎をいくつも解きます。

- **1× が必ずしも悪者ではない**：同期振れ回り自体は材料を疲労させにくい。疲労を生むのは「静止系で一定／低速で変わる」荷重のほう。
- **重力やミスアライメントの定常力は 1× 疲労になる**：これらは静止系で（ほぼ）一定なので、回転系では 1× の交番。横置きロータの宿命です。
- **異方性や安定性の解析が楽になる**：支持が回転系で対称な問題は、回転座標系に乗り換えると方程式が「止まった問題」に化けて、ぐっと見通しがよくなります（軸受やジャイロの安定性で多用される手法です）。

回転座標系は「新しい現象」ではなく、**同じ現象を別の椅子から眺める道具**です。椅子を変えると、静止系では見えなかった交番応力が立ち上がり、見えていた振れ回りが消える。どこに座って見るかを選べることが、ロータダイナミクスの強力な武器になります。

**模式・出典について**：回転座標系での「静止系の一定荷重＝回転系の 1× 交番応力（回転曲げ）」は、回転機械の疲労設計で標準的に使われる古典的な結果です。図は前向き同期振れ回りの模式で、回転系では一点に退化することを示しています。座標変換 $\zeta = z\,e^{-i\Omega t}$ は、その関係を最短で表したものです。
