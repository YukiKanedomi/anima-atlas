# 振幅と位相を1枚に——ボード線図とポーラ線図

回転機械を止まった状態から定格まで上げていく「**昇速試験**」では、各回転数で振動の**大きさ**と**位相**を記録します。この2つをどう見せるかで、診断のしやすさが変わります。代表的なのが**ボード線図**と**ポーラ線図**。同じデータの、違う見せ方です。

## ボード線図：振幅と位相を縦に2段

いちばん素直なのは、横軸に回転数をとって、**振幅**と**位相**を上下に並べること。これがボード線図です。

下で **r（回転数）** を動かしてみてください。カーソルが両パネルを走査します。

- **振幅**は危険速度（$r=1$）で山。第1章の共振曲線そのものです。
- **位相**は、低速の $0°$ から高速の $180°$ へ、$r=1$ で**ちょうど $90°$** を通る S字。第1章「位相の話」で見た遅れが、回転数の関数として見えています。

**位相が $90°$ を切る点が危険速度**——これは振幅の山が多少なまっていても、位相のほうがくっきり危険速度を指してくれる、という現場の知恵につながります。

## ポーラ線図：振幅と位相を1枚に束ねる

ボードが「振幅」と「位相」を別々のグラフにするのに対し、**ポーラ線図**はこの2つを**1枚の複素平面**にまとめます。各回転数での応答を、**長さ＝振幅・向き＝位相**のベクトルと見て、その先端の軌跡を描くのです。

下のポーラ線図の**再生**を押してください。回転数を上げていくと、応答ベクトルが原点から伸びて**ループ**を描きます。そして——**上のボード線図のカーソルも一緒に動きます**。同じ昇速を、2つの目で同時に見ているわけです。

:::

2つは同じ `link` グループでつながっているので、**ポーラの再生でボードのカーソルが走り**、**どちらの ζ スライダーを動かしても両方が変わります**。表現は違っても、中身は一つの応答だと体で分かります。

:::deep ベクトルの長さと向きが、振幅と位相そのもの
ポーラ線図の各点は、複素応答ベクトルの先端です。

$$Z(r) = \frac{r^2}{(1-r^2) + i\,2\zeta r} = M(r)\,e^{-i\varphi(r)}$$

- **長さ** $|Z| = M(r) = r^2/\sqrt{(1-r^2)^2+(2\zeta r)^2}$ ＝ ボード上段の振幅。
- **向き** $-\varphi(r) = -\operatorname{atan2}(2\zeta r,\ 1-r^2)$ ＝ ボード下段の位相（遅れ）。

$r=1$ を入れると $Z = -i/(2\zeta)$。実部ゼロ・真下向きで長さ $1/(2\zeta)$——これがループの底であり、振幅の山であり、位相 $90°$ の点。**3つは同じ瞬間**です。
:::

## どちらを使うか

- **ボード**は「どの回転数で何が起きたか」を読むのに強い。複数の危険速度があるとき、山と位相段差がいくつ並ぶかで本数が分かります。
- **ポーラ**は「危険速度の近くの振る舞い」と**バランシング**に強い。応答ベクトルの向きが、不釣り合いの**重い角度**を直接指すので、どこにおもりを付けるかの判断に直結します（次章の影響係数法へつながります）。

同じ昇速データを、ボードで「時間軸的に」、ポーラで「ベクトル的に」読む——この往復が、現場の振動診断の基本動作です。

**模式・出典について**：両図ともジェフコット不釣り合い応答 $Z(r)=r^2/((1-r^2)+i2\zeta r)$ をそのまま描いた**模式**で、振幅・位相・ループ位置はすべて式どおりです。2図は同一パラメータ（$\zeta$ と掃引中の $r$）を共有しています。
