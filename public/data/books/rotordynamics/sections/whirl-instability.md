# 軸受が振れ回りを発散させる——自励振動とクロスカップリング

これまでの揺れは、不釣り合いという**外からの加振**に対する応答でした。回転数を上げても、共振の山を過ぎれば収まる。でもロータには、もっと厄介な揺れがあります——**外から揺らさなくても、勝手に大きくなっていく**揺れ。これが**自励振動（self-excited vibration）**で、その代表が**オイルホワール**です。

犯人は、軸受やシールの中で生まれる**「変位に直角な力」**です。

ふつうのばねは、変位した向きに**引き戻す**力（直接剛性）を出します。ところが油膜やシール内の流れは、それに加えて**変位と直角な向き**の力を出す。これを**クロスカップリング剛性**と呼びます。この横向きの力が、ロータを**回転と同じ向きに押し続け**、振れ回りにエネルギーを注ぎ込んでしまうのです。

下のブロックで、クロスカップリング **q** と **減衰 c** を動かしてみてください。軌道がどう変わるか見てください。

ポイントは、**減衰 c とクロスカップリング q の綱引き**だということ。

- **q < c**：減衰が勝つ → 軌道は内へ収束。少々つつかれても揺れは収まる（安定）。
- **q > c**：直角な力が勝つ → 軌道は外へ発散。何もしなくても揺れが育つ（**不安定**）。
- 境目は **q = c·ωₙ**。この一線を越えるかどうかが、安定性の分かれ目です。

そして発散する振れ回りは、いつも**前向き**で、振動数は**ほぼ固有振動数 ωₙ**。回転数そのものではありません。だからオイルホワールは、回転数より**遅い**揺れ——第6章のスペクトルで見た **0.43×** の正体がこれです（運転回転数が危険速度の約2倍のとき、ωₙ での揺れは 1× の約半分の位置に立つ）。「1× より低い位置のピーク」を警戒するのは、この自励不安定を見つけるためでした。

## 同じ機構が、いろいろな顔で現れる

クロスカップリング剛性 **q** は、ターボ機械のあちこちで生まれます。**犯人は違っても、数学は同じ**——どれも「q が減衰に勝つと前向きに発散」です。

- **オイルホワール／ホイップ**（すべり軸受）：油膜の周方向の流れが q を生む。最も古典的。
- **シール力**（ラビリンス・バランスピストン）：旋回流（スワール）が q を生む。**スワールブレーキ**で旋回を弱めると安定化する。一方、シールの直接剛性（**Lomakin効果**）はむしろ安定化に働く。
- **アルフォード力／トーマス力**（動翼の先端すきま）：ロータが偏ると、すきまの狭い側の動翼が効率よく働き、**接線方向の力**＝q が出る。段の動力（トルク）に比例するので、**高出力機ほど危ない**。

:::deep なぜ「直角な力」が発散を生むのか
複素座標 $z=x+iy$ で書くと、クロスカップリング $q$ は剛性を**複素数**にします。

$$m\ddot z + c\dot z + (k - i\,q)\,z = 0$$

解を $z=Z\,e^{\lambda t}$（$\lambda=\sigma+i\omega$）と置くと、特性方程式は $m\lambda^2 + c\lambda + (k-iq)=0$。安定限界 $\sigma=0$（$\lambda=i\omega$）を入れると、

- 実部：$-m\omega^2 + k = 0 \Rightarrow \omega=\omega_n$（前向き、固有振動数で振れ回る）
- 虚部：$c\,\omega - q = 0 \Rightarrow q = c\,\omega_n$（安定限界）

$q$ がこれを超えると $\sigma>0$＝振幅が時間とともに $e^{\sigma t}$ で増える。直角な力が1回転ごとに正味の仕事をして、エネルギーを注ぎ込むからです。上の図は、この方程式を実際に数値積分（RK4）して描いています。
:::

## どう抑えるか

自励不安定への対策は、**綱引きを減衰側に傾ける**ことに尽きます。

- **減衰 c を増やす**（ダンパ軸受、スクイーズフィルムダンパ）。
- **クロスカップリング q を減らす**（ティルティングパッド軸受は構造的に q がほぼゼロ。シールのスワールブレーキ）。
- 危険速度や運転点を、不安定が顔を出す回転数から**離す**。

「測って・読んで・直す」が不釣り合いへの対処なら、自励不安定は**そもそも起こさない設計**で対処する——回転機械を高速・高出力にしていくほど、この安定性の見極めが効いてきます。

**模式・出典について**：図は1自由度（複素）のクロスカップリング・モデル $m\ddot z+c\dot z+(k-iq)z=0$ を RK4 で積分した**模式**です。安定限界 $q=c\,\omega_n$、前向き・$\omega\approx\omega_n$ はすべて式どおり。$q$ の物理的な出所（油膜・シール・空力）の係数値までは立ち入らず、機構の共通性を示すことに絞っています。
